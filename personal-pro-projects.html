Fibonacci Calculator

Explore my project showcasing the power of containerization with Docker! This web application calculates Fibonacci values for a given index and utilizes Docker to package and manage its components seamlessly. Docker Containers: Discover the magic of Docker as this project leverages containerization to run PostgreSQL, Redis, API, Client, and Worker components independently.
Efficient Development: Develop and test effortlessly by making changes to the source code within the client, server, and worker directories, with automatic updates reflecting in the Docker containers.
User-Friendly: Use the web interface to enter an index and instantly receive the corresponding Fibonacci value, with a display of calculated values and seen indexes.
Easy Setup: Get started with ease by cloning the repository, running a single docker-compose up command, and accessing the application at http://localhost:3050.
